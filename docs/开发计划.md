| 项目 | 方案 |

| 数据库 | MongoDB + Mongoose |
| 认证 | JWT |
| 图片存储 | 本地存储（后期迁移云存储） |
| 小程序 UI | 原生 + 自定义组件 |
| Flutter 状态管理 | Riverpod |
