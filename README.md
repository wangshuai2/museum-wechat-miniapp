## 🚀 快速开始

### 1. 环境准备

**后端环境**:
```bash
cd src/backend
npm install
cp .env.example .env
# 配置环境变量后启动
npm run start:dev
```

**微信小程序**:
```bash
cd src/frontend
# 使用微信开发者工具打开项目
# 配置 appid 和项目设置
```

**Flutter 客户端**:
```bash
cd src/flutter
flutter pub get
flutter run
```

### 2. 开发流程

```
1. 架构师输出技术方案和接口文档
2. 前后端并行开发
3. 测试助手进行代码审查
4. 修复问题并重新审查
5. 合并代码并部署
```
